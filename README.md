## 🚀 How to go live (one-time setup)

### Step 1 — Merge the Pull Request
1. Go to **[Pull Requests](https://github.com/Korenty/korenty-creatives/pulls)** in this repository.
2. Open the open PR titled **"Build complete Korenty multi-page website"**.
3. Click **"Merge pull request"** → **"Confirm merge"**.

### Step 2 — Enable GitHub Pages
1. Go to **Settings** → **Pages** (left sidebar, under *Code and automation*).
2. Under **Source**, select **"GitHub Actions"** from the dropdown.
3. Click **Save**.

> That's it. GitHub will automatically run the deployment workflow and publish the site.

### Step 3 — View your live site
After the workflow finishes (≈ 1 minute), your site will be live at:

```
https://korenty.github.io/korenty-creatives/
```

You can check the deployment status any time under **Actions** → **Deploy to GitHub Pages**.
